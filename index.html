HTML

    • The index.html file is a common default website's homepage.
    • The top level of html tells the browser things like what type of document <!DOCTYPE html/> and what language <html lang="en">
    • The head element contains information about the webpage, such as metadata and title.
    • Metadata, or meta, helps the browser render the page correctly.
    • The title is important for Search Engine Optimization (SEO), and is displayed in the browser tab.
    • The body element represents the visible content shown to the user.
    • Parent elements hold nested child elements.
    • The header element usually contains navigation and/or intro content.
    • Heading tags from h1 to h6, in order; h1 should only be used once on a page.
    • The img elements include the src attribute for image file location, and alt atribute describes the image and is useful for screen readers.
    • The main element contains the main content of the document.
    • The footer element usually contains info such as author, copyright, contact, sitemap, and navigation.
    • The section elements are semantic elements breaking up the content by theme.
    • The ul element is an unordered list, containing any number of li (list) elements.

CSS

  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • CSS Rule: selector { declaration block }, with declarations inside as such `property: value;`
  • Example CSS Rule: div {color: blue; text-align: justify}
  • CSS box model: Margin, Border, Padding, Content; each is inside the next
  • Margin: space around the *outside* of an element
  • Border: the border
  • Padding: space around the content *inside* the element; adding padding makes the entire element bigger
  • Content: the text
  • A `.` before a selector indicates a class, such as `.card`
